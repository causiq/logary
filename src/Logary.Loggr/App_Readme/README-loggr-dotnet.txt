Please note that in order to complete the installation of loggr-dotnet you will have to do the following:

1. Create a new log on Loggr (or use an existing log)
2. Grab your LogKey and ApiKey from the log's settings
3. Add both keys to your web/app.config files

NOTE: If you're using .NET 4, make sure you are using the full .NET 4 Profile
